# Naukaa – Job Portal Platform

A job portal Platform for Students and Professionals

## Tech Stack

- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB

## Features

- Job Seeker & Employer flows
- Admin Panel
- Job moderation
- Subscription & payments

## Setup

### Frontend

cd frontend
npm install
npm run dev

### Backend

cd backend
npm install
npm run dev
